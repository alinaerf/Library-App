# Online Library application

## Functionality:

* [x]  Login
* [x]  Register
* [x]  Add books
* [x]  Edit information about authors
* [x]  Book recommendations
* [x]  Notification upon new book being added

- React.JS

- Node.JS

- MongoDB

- GraphQL
